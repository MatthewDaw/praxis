# Monica Screenshots

Store dashboard evidence screenshots here when they are small enough to keep in
the repository. Large media should stay outside git and be linked from
`docs/monica/REHEARSAL_LOG.md`.

Recommended capture set:

- Data-source banner or control state.
- Candidate list with provenance.
- Candidate detail confidence breakdown.
- Promote confirmation dialog showing `proposed -> active`.
- Contradiction resolution before and after.
- Eval metrics or eval-status panel.
